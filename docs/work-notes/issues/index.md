# 이슈 모음

실무에서 마주한 다양한 이슈들과 그 해결 과정을 기록합니다.

## 이슈 기록의 목적

- 동일한 문제 재발생 시 빠른 해결
- 팀원들과 문제 해결 경험 공유
- 문제 해결 패턴 학습

## 이슈 카테고리

여기에 이슈들을 카테고리별로 정리할 수 있습니다:
- 성능 이슈
- 버그 수정
- 배포 관련 이슈
- 브라우저 호환성
- 보안 이슈

## 이슈 작성 템플릿

```markdown
### 문제 상황
무엇이 문제였는가?

### 원인 분석
왜 발생했는가?

### 해결 방법
어떻게 해결했는가?

### 배운 점
다음에는 어떻게 예방할 수 있는가?
```
